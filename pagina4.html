Proyecto: Tarea1 - Diseño Computacional y Programación Web

Descripción del Proyecto

Este proyecto, Tarea1, forma parte de la clase de Diseño Computacional y Programación Web. Su objetivo principal es aplicar los conocimientos adquiridos sobre HTML para construir páginas web estructuradas y funcionales. La tarea implica crear varias páginas que demuestren el uso adecuado de encabezados, listas, enlaces y otros elementos fundamentales en el diseño web.

A través de este proyecto, se busca fortalecer las habilidades de diseño de estructuras web, practicar la organización de contenido y establecer las bases para la incorporación futura de estilos y funcionalidades interactivas.

Objetivos del Proyecto

  • Desarrollar una comprensión sólida de la estructura de HTML y sus elementos básicos.
  • Practicar la organización y jerarquía de contenido en una página web.
  • Aplicar listas, encabezados y secciones para una presentación clara y profesional del contenido.
  • Preparar el diseño para la adición de CSS y JavaScript en futuras tareas.

Tecnologías Utilizadas

  • HTML5: Para la creación de la estructura de la página web.
  • Editor de texto (como Visual Studio Code): Para el desarrollo del código.
  • Navegador web: Para la visualización y pruebas de la página.
